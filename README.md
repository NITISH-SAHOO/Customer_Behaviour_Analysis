# Customer_Behaviour_Analysis
Data Analytics project showcasing Customer Shopping Behaviour Analysis using Python, SQL &amp; Power BI.

📊 Data Analytics Project: Customer Shopping Behaviour Analysis
📌 Overview
This project focuses on analyzing customer shopping behavior using a complete data analytics pipeline. It involves data loading, exploratory data analysis (EDA), data cleaning, SQL-based data handling, and building an interactive dashboard in Power BI. The project also includes a report and presentation for effective communication of insights.

📂 Dataset
The dataset contains customer shopping-related information, including:
Customer Age
Purchase Category
Purchase Amount
Purchase Frequency
Review Ratings
Discount Applied
🛠️ Tools & Technologies
Python (Pandas, NumPy) – Data cleaning & EDA
SQL (MySQL/PostgreSQL/SQL Server) – Data storage & querying
Power BI – Dashboard & visualization
Jupyter Notebook – Development environment
Gamma – Presentation (PPT creation)

🔄 Steps Performed
1. Data Loading
Imported dataset into Python using Pandas
Inspected structure using .info() and .describe()
2. Exploratory Data Analysis (EDA)
Analyzed distributions and trends
Identified missing values and inconsistencies
Explored relationships between variables
3. Data Cleaning
Handled missing values using appropriate techniques
Renamed columns for consistency
Removed duplicate/redundant columns
Created new features (e.g., age groups, frequency mapping)
4. SQL Integration
Imported cleaned data into SQL database
Performed queries for structured data analysis
Enabled connection with Power BI
5. Dashboard Creation (Power BI)
Connected Power BI with SQL database
Built interactive visuals and KPIs

📊 Dashboard
The Power BI dashboard includes:
Sales analysis by category
Customer segmentation by age group
Purchase frequency trends
Discount impact analysis
Key KPIs (Total Sales, Average Rating, etc.)

📈 Results
Identified key customer behavior patterns
Found impact of discounts on purchasing decisions
Segmented customers for better targeting
Generated insights for business decision-making

▶️ How to Run
Step 1: Python (EDA & Cleaning)
Open the Jupyter Notebook and run all cells
Step 2: SQL
Import cleaned dataset into MySQL/PostgreSQL/SQL Server
Run SQL queries if required
Step 3: Power BI
Connect Power BI to the SQL database
Load dataset and open the dashboard file (.pbix)
